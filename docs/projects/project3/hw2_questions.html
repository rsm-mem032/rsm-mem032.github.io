Replication of Card and Krueger (1994)

Minimum Wages and Employment: A Case Study of the Fast-Food Industry in New Jersey and Pennsylvania

Author

Mengyao Ma

Published

April 22, 2026

Introduction

Card and Krueger (1994) study whether the April 1, 1992 minimum wage increase in New Jersey (from $4.25 to $5.05) reduced fast-food employment. They compare fast-food stores in New Jersey to stores in eastern Pennsylvania, where the minimum wage did not change.

The core empirical design is a difference-in-differences (DID): compare the change in employment in New Jersey before vs. after the policy to the same before-after change in Pennsylvania.

Note on replication targets: the assignment emphasizes matching point estimates closely. Standard errors can differ across replications because of choices about sample restrictions and handling of temporarily closed stores.

Main Analysis

# Load and label the public data
path <- "data/raw/public.dat"
raw <- read.table(path, header = FALSE, na.strings = ".", strip.white = TRUE, fill = TRUE)

names(raw) <- c(
  "sheet","chain","co_owned","state","southj","centralj","northj","pa1","pa2","shore",
  "ncalls","empft","emppt","nmgrs","wage_st","inctime","firstinc","bonus","pctaff","meals",
  "open","hrsopen","psoda","pfry","pentree","nregs","nregs11","type2","status2","date2",
  "ncalls2","empft2","emppt2","nmgrs2","wage_st2","inctime2","firstin2","special2","meals2",
  "open2r","hrsopen2","psoda2","pfry2","pentree2","nregs2","nregs112"
)

d <- raw

d$nj <- d$state
d$emptot <- d$empft + 0.5 * d$emppt + d$nmgrs
d$emptot2 <- d$empft2 + 0.5 * d$emppt2 + d$nmgrs2
d$demp <- d$emptot2 - d$emptot
d$closed <- d$status2 == 3

d$pmeal <- d$psoda + d$pfry + d$pentree
d$pmeal2 <- d$psoda2 + d$pfry2 + d$pentree2

d$chain_bk <- as.integer(d$chain == 1)
d$chain_kfc <- as.integer(d$chain == 2)
d$chain_roys <- as.integer(d$chain == 3)
d$chain_wendys <- as.integer(d$chain == 4)

# GAP variable used in Card-Krueger regression specs
d$gap <- NA_real_
d$gap[d$state == 0] <- 0
idx <- which(d$state == 1 & !is.na(d$wage_st) & d$wage_st >= 5.05)
d$gap[idx] <- 0
idx <- which(d$state == 1 & !is.na(d$wage_st) & d$wage_st > 0 & d$wage_st < 5.05)
d$gap[idx] <- (5.05 - d$wage_st[idx]) / d$wage_st[idx]

mean_se <- function(x) {
  n <- sum(!is.na(x))
  c(mean = mean(x, na.rm = TRUE), se = sd(x, na.rm = TRUE) / sqrt(n))
}

Something Additional (Difference-in-Differences Visualization)

I add a DID figure to visualize the main identification pattern by comparing New Jersey and Pennsylvania before and after the policy change.

# Balanced sample for a clean pre/post comparison
bal <- subset(d, !is.na(emptot) & !is.na(emptot2))

mean_pre_nj <- mean(bal$emptot[bal$nj == 1])
mean_post_nj <- mean(bal$emptot2[bal$nj == 1])
mean_pre_pa <- mean(bal$emptot[bal$nj == 0])
mean_post_pa <- mean(bal$emptot2[bal$nj == 0])

plot(
  x = c(1, 2), y = c(mean_pre_pa, mean_post_pa),
  type = "b", pch = 16, lwd = 2, col = "steelblue",
  xaxt = "n", xlab = "Survey Wave", ylab = "Average FTE Employment",
  ylim = range(c(mean_pre_nj, mean_post_nj, mean_pre_pa, mean_post_pa)),
  main = "Difference-in-Differences Pattern: NJ vs PA"
)
axis(1, at = c(1, 2), labels = c("Before (Wave 1)", "After (Wave 2)"))
lines(c(1, 2), c(mean_pre_nj, mean_post_nj), type = "b", pch = 17, lwd = 2, col = "firebrick")
legend("topright", legend = c("Pennsylvania", "New Jersey"), col = c("steelblue", "firebrick"), pch = c(16, 17), lwd = 2, bty = "n")

Interpretation: PA employment declines between waves, while NJ employment rises slightly. The difference in these trends is the DID estimate, consistent with the paper’s conclusion that the minimum wage increase did not reduce fast-food employment in this case.
